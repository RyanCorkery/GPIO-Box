User Instructions:

The following will save the HTML file to the arduino SD card

1) Edit Main.html to whatever you need
2) Copy Main.html contents
3) Open html.txt
4) Delete contents
5) Paste content from Main.html
6) Save html.txt
7) Open and run HTML_to_txt.cpp
8) Open html_Arduino.txt and copy content
9) Open HTML_Uploader arduino file
10) Paste html_Arduino.txt contents in arduino sketch where indicated
11) Upload sketch to arduino
